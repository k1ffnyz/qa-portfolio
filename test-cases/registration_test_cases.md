# Test Cases — User Registration

## TC-REG-001 — Успешная регистрация
**Предусловие:** пользователь не зарегистрирован

**Шаги:**
1. Открыть страницу регистрации
2. Ввести корректный email
3. Ввести корректный пароль
4. Нажать «Зарегистрироваться»

**Ожидаемый результат:**  
Пользователь успешно зарегистрирован

---

## TC-REG-002 — Некорректный email

**Шаги:**
1. Ввести email с недопустимыми символами
2. Нажать «Зарегистрироваться»

**Ожидаемый результат:**  
Отображается сообщение об ошибке
